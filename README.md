# Portfolio Website

Modern Full Stack Portfolio Website built with React, TypeScript, Redux Toolkit, React Query, Node.js, Express and MongoDB. | React, TypeScript, Redux Toolkit, React Query, Node.js, Express va MongoDB yordamida yaratilgan zamonaviy Full Stack Portfolio veb sayti.

## Features | Imkoniyatlar

* Responsive Design | Responsive dizayn
* Dark / Light Mode | Qorong'u va Yorug' rejim
* Multi Language Support (EN / UZ / RU) | Ko'p tillilik (EN / UZ / RU)
* Dynamic Projects from API | API orqali dinamik loyihalar
* Project Search | Loyiha qidiruvi
* Project Details Page | Loyiha tafsilotlari sahifasi
* Contact Form | Kontakt formasi
* Skeleton Loading | Skeleton yuklanish effekti

* Scroll To Top | Tepaga qaytish tugmasi
